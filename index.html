Shoot-on-the-Move

Targeting system · design & implementation

A turretless robot normally stops to line up each shot. I built a system that scores while driving: robot velocity offsets the aim point, an iterative solver leads the goal using flight-time data I measured, and the result is a 90%+ scoring rate on the move.

Unity Robotics Simulator

CloSim · simulation & driver-practice tooling

I brought our team's robot model into CloSim, a Unity-based FRC simulator, so we can rehearse autonomous routines and strategy before the real robot exists. I set up the robot's build frame and worked with its auto-aim targeting in the sim, plus multiplayer testing so drivers and programmers can practice together.

Computer vision · PID

Vision auto-alignment

Integrated Limelight vision data with PID controllers so the robot squares to the goal on its own, cutting alignment time out of every scoring cycle. Tuned the loops to settle fast without overshoot.
